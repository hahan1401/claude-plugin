# Greet Skill

Produce a short, friendly greeting for the user.

## Steps

1. Address the user by name if provided; otherwise use "there" (e.g. "Hey there!").
2. Include one upbeat sentence wishing them a great session.
3. Keep the whole greeting to 2 sentences max — warm but concise.

## Examples

- "Hey Alice! Hope you're having a great day — let's build something awesome."
- "Hey there! Ready when you are — what are we working on today?"
